# Magic 8 Ball

Приложение "Магический шар 8". При нажатии на кнопку появляется случайный предсказательный ответ.

## Функционал

- 5 вариантов ответов
- Случайный результат при каждом нажатии

## Технологии

- Swift
- UIKit
- UIImageView
- randomElement()
